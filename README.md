# Spread-Time-CDMA-Transceiver-Digital-Blocks
BSc thesis project implementing the digital baseband blocks of a Spread-Time CDMA transceiver on Xilinx Spartan-6 and Zynq-7000 FPGAs using Verilog. Includes TX/RX architectures, timing control, encoding/decoding, and FPGA-ready implementations adapted for ASIC-oriented development.
